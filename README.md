**프로젝트 소개**

</aside>

도깨비 저택을 탈출하는 **1인칭 퍼즐** 게임

**DirectX11**을 활용한 **그래픽스 엔진 개발** 담당


**담당 업무**

</aside>

**DirectX11, Assimp**를 활용한 Resource Manager

재질에 따른 빛의 반사를 모방한 **Physically Based Rendering (PBR)**

실내 조명을 표현하기 위한 **Point Light**

현실의 깊이를 모방한 **Shadow Mapping**
